nekStab¶

Synopsis¶

nekStab (Stability for Nek5000) is an open-source toolbox for massively-parallel eigenproblem and base flow computation developed for the Nek500 solver. It aims at solving any fluid flow of an incompressible, Newtonian fluid that can benefit from a Krylov-Schuer eigensolver.
